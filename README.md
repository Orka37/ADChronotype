# ADChronotype

ADChronotype is a web-based cognitive risk assessment tool that estimates Alzheimer’s disease likelihood based on sleep chronotype, sleep duration, demographic factors, and cognitive test modeling.

This application was developed for neuroscience and machine learning research competitions including CSRSEF and WSSEF.

## Features
- Interactive data input interface
- Cognitive risk modeling
- Machine learning prediction pipeline
- Streamlit-powered web interface
